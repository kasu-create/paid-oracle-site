# まずここを読む

このフォルダは **デスクトップの `paid-oracle-site`** に入っています。

## Cursor で開く

1. **ファイル → フォルダーを開く**
2. **デスクトップ** の **`paid-oracle-site`** を選ぶ（この中に `oracle` フォルダがある階層）
3. 左の一覧に `README.md` や `oracle` が出ればOK

## プレビュー

ターミナルでこのフォルダに移動してから:

```bash
npx --yes serve .
```

ブラウザで `http://localhost:3000/oracle/reading/`

このフォルダで `git init` → リポジトリに push。
