このページについて

URL に /oracle/ を含め、他の無料LPや別プロダクトと導線を分けています。 実際の鑑定画面は /oracle/reading/ です（購入後にお渡しするリンクから開いてください）。UIは GitHub風の作業画面にして、他LPと見た目を分けています。

購入・決済

STORES の商品ページ URL をここに差し込みます。

STORES で購入する （準備中：href を本番URLに置き換え）

購入後の流れ

  1. 決済完了後、メールまたは STORES の案内で 鑑定ページのURL をお渡しします。
  2. URL は /oracle/reading/?token=… 形式を想定（トークンは1回の鑑定ごとに異なります）。
  3. 同じURLで結果の再表示ができる想定です（仕様は実装時に確定）。
